TDW-1148 Hands-on Lab: Building Mobile Applications Using JAXRS, Dojo, and PhoneGap

This is a sample auto insurance PhoneGap application that integrates with the IBM WebSphere application server to call a JAX-RS REST service.

Authors

Bryce A. Curtis, Mobile & Emerging Technologies,
Todd Kaplinger, Web2/FeP Release Architect

Impact 2011 Lab

During this lab you will be writing the Mysurance auto insurance application. It consists of two parts:

  1. Server JAX-RS services running on the IBM WebSphere server
  2. Mobile PhoneGap client running on the Android emulator

Server Code

The lab uses mysurance.war file that needs to be installed in your server project and run on WebSphere. You can download this file using the following link and add it to your project.

  • mysurance.war
